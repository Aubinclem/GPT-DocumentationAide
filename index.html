Garry's Mod

  • 🔧 Sauvegarde JSON pour DarkRP
    Utilisez ce format pour enregistrer les données en JSON si SQL bug : 
    {"steamid": "STEAM_0:0:123456", "cases": [1, 2, 3]}
  • 📦 Créer un addon simple
    Structure typique : 
    myaddon/
├── lua/
│   └── autorun/
│       └── myaddon_init.lua
└── addon.txt
  • 👮‍♂️ ULX ne fonctionne pas
    Assurez-vous que le dossier `ulx` est dans `garrysmod/addons/` et que le fichier `ulx.lua` est bien lu.
  • 💾 Sauvegarde d'entités personnalisées
    Vous pouvez utiliser `file.Write("entites.json", util.TableToJSON(entTable))` côté serveur pour enregistrer l'état de vos entités.
  • 🔐 Gérer les permissions d'outils
    Utilisez `tool.Allow` dans un hook `PlayerGiveSWEP` pour restreindre les outils à certaines équipes.

Xbox

  • 🌐 Problèmes de connexion Xbox Live
    Redémarre ta box, puis va dans les paramètres réseau et utilise "Test de connexion" pour voir les erreurs exactes.
  • 🧾 Résilier un abonnement Game Pass
    Va sur account.microsoft.com/services et clique sur "Gérer" à côté de l'abonnement.
  • 🎮 Partage de jeux entre comptes
    Utilise la fonction "Xbox principale" pour partager tes jeux et abonnements avec un deuxième compte sur la même console.
  • 🕹️ Manette non détectée
    Vérifie les piles, réinitialise la manette (bouton à l’arrière), puis reconnecte-la via USB ou sans fil.

Autres Problèmes

  • 🪟 Réparer une application qui crash sur Windows
    Ouvre les paramètres > Applications > clique sur l’app > Options avancées > Réparer.
  • 📁 Afficher les extensions de fichiers
    Explorateur de fichiers > Affichage > cocher "Extensions de noms de fichiers".
  • 🔊 Son ne fonctionne plus
    Vérifie le périphérique de sortie dans les paramètres Windows > Système > Son.
  • 🔒 Réinitialiser un mot de passe oublié
    Sur Windows : Écran de connexion > "J’ai oublié mon mot de passe" > suivez les étapes via votre compte Microsoft.
